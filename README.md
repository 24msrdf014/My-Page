# 🧑‍💻 Insider Threat Simulation with Behavior-Based Defense

## 📌 Project Overview

This project presents a **GUI-based Insider Threat Simulation System** developed in Python using **PyQt5**, designed as part of the course *Digital Forensics & Information Security*. It emulates a **Cyber Range as a Service (CRaaS)** environment by simulating real-world insider attack scenarios and implementing automated defense mechanisms.

---

## 🎯 Objective

The primary objective of this project is to:

* Simulate **normal and malicious insider activities**
* Detect anomalies using **behavior-based analysis**
* Trigger **automated defensive responses**
* Provide **real-time monitoring through a professional dashboard**
* Generate logs for **digital forensic investigation**

---

## ⚙️ Key Features

* 🖥️ **Interactive PyQt Dashboard** for live monitoring
* 📊 **Real-time activity logging and visualization**
* 🧠 **Anomaly detection using statistical (Z-score) techniques**
* 🚨 **Automated alert generation and risk scoring**
* 🔒 **Defense mechanisms** (account lock, access restriction)
* 🗂️ **SQLite-based logging system** for forensic analysis

---

## 🛡️ Attack–Defense Simulation

The system simulates insider threat behaviors such as:

* Unauthorized access to restricted resources
* Bulk data downloads (data exfiltration)
* Abnormal usage patterns

These behaviors are analyzed against baseline user activity, and **anomalies are detected dynamically**, triggering defensive actions.

---

## 🔍 Forensic Relevance

The project generates valuable artifacts including:

* User activity logs
* Alert and incident records
* Risk score evolution

These logs can be used for:

* **Incident reconstruction**
* **Timeline analysis**
* **Evidence generation in digital investigations**

---

## 🧩 Technologies Used

* **Python** (Core development)
* **PyQt5** (GUI development)
* **Pandas & NumPy** (data analysis)
* **Matplotlib** (visualization)
* **SQLite** (logging and storage)

---

## 🚀 Conclusion

This project demonstrates how **behavior-based detection systems** can effectively identify insider threats, going beyond traditional signature-based approaches. It also highlights the importance of **real-time monitoring, automated defense, and forensic readiness** in modern cybersecurity systems.
